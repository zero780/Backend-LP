# Backend-LP
## Proyecto de 2do parcial LP 

### Necesario instalar:<br />
<ul>
<li>$ python -m pip install Django </li>
<li>$ python -m pip install Pillow</li>
<li>$ python -m pip install django-cors-headers</li>

</ul>

### Correr servicio:
```bash
$ python manage.py runserver
```
